SVN-Searcher by nuncanada

Java project based on Lucene to allow text search of SVN repositories

This is a fork of the project SVN Searcher which can be found at sourceforge: http://svn-search.sourceforge.net/ The project has no commits since 2007 and seems abandoned. I have adapted it for current best practices with Maven and made some small changes to the code. I plan on supporting this fork and feel free to fork and hack away any needed change.
